What is missing for 2026-level risk context

The dataset omits several risk dimensions commonly needed for a modern cardiovascular risk profile.

Conclusion

This project shows how structured clinical variables can support exploratory classification, but it also shows why clinical ML cannot be evaluated only by predictive metrics.

Analytical conclusion

The most useful outcome is not a diagnostic model. It is a transparent workflow that connects clinical patterns, model quality, missing risk dimensions and responsible limitations.
